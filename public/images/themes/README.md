# Archivos de Temas Estacionales

Esta carpeta contiene los logos y fondos para cada tema estacional.

## Estructura de Archivos

### Tema Default (Predeterminado)
- **Logo**: `logo-default.png`
- **Fondo**: `bg-default.jpg`

### Tema Halloween
- **Logo**: `logo-halloween.png`
- **Fondo**: `bg-halloween.jpg`

### Tema Christmas (Navidad)
- **Logo**: `logo-christmas.png`
- **Fondo**: `bg-christmas.jpg`

## Especificaciones

### Logos
- **Formato**: PNG (con transparencia)
- **Tamaño recomendado**: 200x60px
- **Uso**: Se muestra en el header de la página principal y del dashboard

### Fondos
- **Formato**: JPG o PNG
- **Tamaño recomendado**: 1920x1080px (Full HD)
- **Uso**: Fondo del contenedor principal del dashboard (envuelve balance, asesora, bienvenida, slider)

## Cómo Usar

1. Coloca los archivos en esta carpeta con los nombres exactos indicados arriba
2. Activa el tema desde el panel de administración
3. Los cambios se aplican automáticamente

## Notas

- Los archivos deben tener los nombres exactos para que funcionen
- Si un archivo no existe, se mostrará el logo/fondo por defecto
- Los cambios son visibles inmediatamente para todos los usuarios
