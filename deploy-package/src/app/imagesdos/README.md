# Flasti AI - Image Generator

This is the deployment package for the Flasti AI Image Generator web application. To deploy this application to Hostinger:

1. Log in to your Hostinger control panel
2. Navigate to File Manager
3. Go to the public_html directory
4. Upload all the contents of this folder

## Important Notes

- Make sure all files maintain their original names and structure
- The application requires an active internet connection to function
- The API key is included in the script.js file

## Files Included

- index.html - Main HTML file
- styles.css - Stylesheet
- script.js - Application logic
- logo.svg - Logo image
